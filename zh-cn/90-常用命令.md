# 常用命令

## 仓库相关命令

### 查看所有chart仓库列表
```
helm list
```
### 添加chart仓库
```
helm repo add 仓库名称 --username 用户名 --password 密码 chart库地址
```

### 拉取仓库指定版本chart包
```
helm pull 仓库名称/chart包名称 --version 版本 // 更多参数看官网
```

### 打包chart
```
helm package chart包目录地址 --version chart版本
```

### 推送chart包到远程仓库
```
helm push chart包名称 --username 用户名 --password 密码 远程仓库地址
```
